# 🐋 Onchain Wallet Watcher

A simple tool to monitor transactions of selected Ethereum wallets.

## Features

- Track latest transactions of any wallet
- Useful for following whales or smart money
- Lightweight and easy to use
- Supports multiple addresses

## Requirements

- Python 3
- Etherscan API key

## Setup

1. Get a free API key from Etherscan  
2. Add wallet addresses to `wallets.txt`  
3. Insert your API key in `watcher.py`

## Usage

```bash
python watcher.py
